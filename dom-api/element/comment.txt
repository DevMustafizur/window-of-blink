##  Element কি? (DOM এ)

 Element হলো DOM tree-এর সেই অংশ যেটা HTML tag কে represent করে।

 -  HTML এর প্রতিটা visible tag-ই Element

---

##  Element কী কী হতে পারে?

- `<div>`
- `<p>`
- `<button>`
- `<img>`
- `<h1> - <h6>`
- `<span>`
- `<a>`
- `<ul>`, `<li>`
- `<form>`, `<input>`
- `<section>`, `<article>`

---

##  DOM এ Element কোথায় থাকে?

Document → Node → Element

element হলো Node এর একটি special type  
কিন্তু সব Node Element না (Text, Comment থাকে)
