## Event Object Methods (Full list)

- event.preventDefault() - default action বন্ধ করে
- stopPropagation() -event bubbling/capturing flow বন্ধ করে
- stopImmediatePropagation() - একই element এর অন্য listeners-ও বন্ধ করে



-----------


## Common Event Properties (সব event এ থাকে)
type → event name (click, input, keydown)
target → আসল element যেখানে event হয়েছে
currentTarget → যেই element এ listener আছে
bubbles → event bubble করবে কিনা
cancelable → preventDefault করা যাবে কিনা
defaultPrevented → default action বন্ধ হয়েছে কিনা
eventPhase → capture / target / bubble phase
timeStamp → event কখন হয়েছে (ms)
